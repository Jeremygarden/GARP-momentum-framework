# GARP 观察池入选规则 v1.1

> 版本：v1.1 | 制定日期：2026-04-06 | 负责人：Investor Agent
> 本文档定义观察池（Watch Pool）的入选、维持、升级和移除机制，是核心池 Top15 的配套治理规则。

---

## 一、观察池的定位

观察池不是交易信号池，而是**"值得花精力深度研究的预备名单"**。

| 池子 | 定位 | 容量 |
|------|------|------|
| 核心池 Top15 | 当前最佳持仓标的 | 15只 |
| 观察池 Watch | 来自 Top15 降级 或 候补池晋升的高潜力标的 | 5-10只 |
| 候补池 Pipeline | 初步筛选通过、等待事件验证的储备标的 | 15-20只 |

观察池标的需**双周复查**一次，达到升级条件即可替换 Top15 末位。

---

## 二、入选观察池的两条路径

### 路径 A：Top15 → 观察池（降级保留）

路径 A 分两层：**A1 急性冲击**（单条触发，立即降级）和 **A2 慢性恶化**（预警后双周确认再降级）。

---

#### A1：急性冲击 → 立即降级

满足任一条件，当周执行降级，移入观察池。

**动量 / 价格类：**
- M 分（价格动量）单周骤降 ≥ 20 分
- 财报后股价单日跌幅 ≥ 15%
- 季报净利润增速环比下滑 ≥ 30%，且低于市场预期 20% 以上

**风险突变类：**
- 重大诉讼 / 监管调查，涉及金额 > 公司市值 5%
- CEO / CFO / 创始人非正常离职（非退休、非任期届满）
- 地缘政治或政策黑天鹅直接打击核心业务（制裁、集采覆盖核心产品等）

---

#### A2：慢性恶化 → 预警后降级

触发以下任一条件时，**不立即降级**，而是在 Bitable 中打上 `⚠️ 预警` 标签，并在下次双周复查时重点核查。若复查时问题仍未改善或进一步恶化，则执行降级。

**质量因子恶化（Q分）：**
- ROE 连续两季下行，且累计跌幅 > 3pct
- 毛利率连续 3 季同比下滑，每季 > 2pct
- 应收账款周转率单季同比恶化 > 20%
- 经营现金流连续两季为负（利润无法变现）

**成长性预期折损（G分）：**
- 市场一致预期净利润被机构下调 > 15%（未来12个月）
- 季报净利增速连续两季环比收窄 > 10pct
- 核心业务收入增速跌破 20% 且无新业务弥补

**估值泡沫化（V分）：**
- PEG 单季从 <1 升至 >2.0（股价涨速远超业绩）
- PE 超过近 3 年历史 90% 分位
- V 分（估值合理性）跌破 50 分

**竞争格局恶化：**
- 核心产品市占率单季下滑 > 2pct
- 同赛道竞争对手净利增速连续两季超本标的 > 30pct
- 重大新技术出现且公司无应对布局（技术路线颠覆风险）

**股东行为异常：**
- 大股东 / 实控人减持超过总股本 5%

---

**降级后的维持条件：**
- 基本面（G/Q）无实质性恶化
- 若 G/Q 同时崩坏，直接移除，不保留观察池席位

---

### 路径 B：候补池 → 观察池（事件晋升）

候补池标的触发以下**强信号（单独触发）** 或 **弱信号（任意两个叠加）**，可晋升观察池。

#### 🔴 强信号（单独触发，置信度高）

| 信号 | 具体标准 | 备注 |
|------|----------|------|
| 业绩超预期 | 业绩预告/快报净利润超一致预期 ≥ 20% | G 分实质性重估 |
| 重大合同 | 单笔合同金额 > 公司市值 5% | 增速预期修正 |
| 主力资金异动 | 主力净流入单日 > 30 日均值 3 倍，且同日涨幅 > 3% | 须量价双确认 |

#### 🟡 弱信号（需任意两个叠加触发）

| 信号 | 具体标准 | 备注 |
|------|----------|------|
| 量价双确认异动 | 成交量 > 20日均量 2.5倍，收阳线，上影线 < 实体 1/2 | 排除放量跌 |
| 赛道ETF背离 | 所在行业ETF单周涨幅 > 8%，但个股滞涨 > 3% | 补涨机会信号 |
| 机构排名跃升 | 机构净买入排名**单日进入全市场前50**，或**周内排名上升 ≥ 10位** | 须是机构而非散户 |
| 估值超跌错杀 | PEG < 0.3，且近30日跌幅 > 15%，ROE无恶化 | 价值回归型机会 |
| 产业链传导 | 同赛道龙头大涨后48h内，供应链/受益标的尚未启动 | 需判断传导逻辑成立 |
| 上调评级+目标价 | 至少2家机构上调至买入，目标价均高于当前价20%+ | 机构一致预期拐点 |

## 三、观察池的维持与移除

### 双周复查清单

每两周对观察池标的进行以下检查：
1. M 分变化（技术面是否修复）
2. 最新季报/月度数据（G 分是否稳定）
3. ROE 趋势（Q 分是否恶化）
4. 升级条件是否已满足
5. 是否有新的风险事项

### 移除条件（满足任一即移除）

- ROE 跌破 12%（实质性基本面恶化）
- 净利润连续两季负增长
- 行业政策风险重大变化（监管打压、集采等）
- 在观察池超过 **3个月** 仍未满足升级条件
- 已晋升核心池 Top15

---

## 四、升级核心池的条件

从观察池晋升 Top15，须**同时满足**：

| 条件 | 要求 |
|------|------|
| M 分 | ≥ 60（路径A降级后修复）或 ≥ 75（路径B晋升） |
| 技术面 | RSI 企稳 + MACD 金叉（右侧确认） |
| 基本面验证 | 最新季报净利润增速符合入池预期 |
| 估值 | PEG < 1.5 |
| 替换逻辑 | 综合评分须高于 Top15 末位标的 |

## 五、Bitable 观察池字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| Name | 文本 | 股票名称 |
| 股票代码 | 文本 | 代码 |
| 市场 | 单选 | A股/港股 |
| G分 | 数字 | 成长性评分 0-100 |
| Q分 | 数字 | 质量因子评分 0-100 |
| 当前总分 | 数字 | 综合评分 |
| 纳入原因 | 单选 | M崩/R风险/暴跌待确认/季报待验证/PEG偏高监控 |
| **事件触发类型** | 多选 | 业绩超预期/重大合同/主力资金异动/量价双确认/赛道ETF强个股滞涨/机构排名跃升/PEG超跌错杀/产业链传导/M分骤降降级/候补池晋升 |
| **触发信号强度** | 单选 | 🔴强信号单独触发 / 🟡弱信号×2叠加 / ⬇️Top15降级 / ⬆️候补池晋升 |
| 待验证事项 | 文本 | 具体待观察的基本面问题 |
| 技术状态 | 单选 | 等待右侧/RSI企稳/MACD金叉/量价共振/已确认入场 |
| **资金流异动** | 文本 | 触发时的资金流数据（主力净流入/均值倍数/当日涨幅） |
| **估值锚状态** | 单选 | 🟢PEG<0.3超跌错杀 / 🟡PE在近3年10%分位 / ⚪正常区间 / 🔴PE超30x偏贵 |
| **赛道景气信号** | 文本 | ETF背离、产业链传导等具体描述 |
| 升级条件 | 文本 | 个股定制的晋升 Top15 具体条件 |
| **下次事件监控点** | 文本 | 下次需关注的具体时间点或事件（季报日、合同兑现窗口等） |
| 纳入日期 | 日期 | 进入观察池的日期 |
| 下次双周复查日 | 日期 | 进入日期 +14天自动计算 |
| 状态 | 单选 | 监控中/升级核心池/降级候补池/已剔除 |

**加粗字段** = 本次 v1.0 新增
