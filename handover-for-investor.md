## 二、已完成工作 · 成果与结论

### 2.1 投资框架：GARP + 质量动能 v2.0

**核心理念：**
GARP（Growth at a Reasonable Price）= 在合理估值下买入具备持续成长性和高质量的股票，叠加动量因子过滤短期机会。区别于纯价值/纯成长，是中间道路，适合 A股/港股中期持仓逻辑。

**四维评分体系（各25分，满分100）：**

| 维度 | 代号 | 核心指标 | 高分标准 | 低分标准 |
|------|------|---------|---------|---------|
| 成长性 | G | 未来3年净利润CAGR | ≥30% 满分，≥15% 及格 | <10% 或负增长 |
| 估值合理性 | A | PEG、前向PE | PEG<0.5 满分，PEG<1 及格 | PEG>1.5 或PE>40x |
| 质量因子 | Q | ROE、现金流、负债率 | ROE>25% 且持续改善 | ROE<10% 或现金流恶化 |
| 价格动量 | M | 近期股价强度、催化剂 | 创250日新高+事件驱动 | 近期暴跌或无动量 |

**核心筛选门槛（GARP标准过滤器）：**
- ✅ 净利润连续3年以上为正（剔除亏损年份）
- ✅ 未来3年净利润CAGR > 15%
- ✅ 前向PE < 25x（或 PEG < 1 优先）
- ✅ ROE > 12%（连续改善趋势）
- ✅ 资产负债率 < 60%（排除金融股/高杠杆周期股）
- ❌ 排除：纯周期股、银行/保险（高杠杆结构）、项目制/融资依赖型

**PEG 计算公式：**
```
PEG = 前向PE(x) ÷ 未来3年净利润CAGR(%)
理想区间：< 0.5（极度低估）
安全区间：0.5 ~ 1.0
高估警示：> 1.0
```

### 4.2 Skill 核心功能模块

**模块一：Universe 筛选（初步过滤）**
```
输入：市场范围（A股/港股/全市场）、行业范围（可选）
操作：
  1. 排除亏损年份 > 1次（过去12个季度）
  2. 排除负债率 > 60%（金融股/高杠杆）
  3. 排除纯周期/项目制行业（煤炭/钢铁/房地产/公用事业/银行）
  4. 过滤 ST股、停牌股、次新股
输出：候选股票池
```

**模块二：四维打分（核心）**
```
G - 成长性评分（0~25）：
  - 收集：过去3年净利润增速、未来3年CAGR预期（Wind一致预期）
  - 规则：CAGR≥30% → 25分；≥20% → 20分；≥15% → 15分；<10% → 5分以下

A - 估值合理性评分（0~25）：
  - 收集：前向PE（2026E）、PEG
  - 规则：PEG<0.5 → 25分；<0.8 → 22分；<1.0 → 18分；>1.5 → 10分以下

Q - 质量因子评分（0~25）：
  - 收集：ROE（TTM+预测趋势）、经营现金流净额/净利润比、资产负债率
  - 规则：ROE>30%且改善 → 25分；>20% → 20分；>12% → 15分；<10% → 8分以下

M - 价格动量评分（0~25）：
  - 收集：近6个月股价相对沪深300超额收益、是否创250日新高、最近重大催化剂
  - 规则：超额>50% + 创新高 + 事件驱动 → 25分；超额>20% → 18分；负超额 → 5分以下
```

**模块三：数据获取策略**
```
优先顺序：
1. Wind/Choice API（最权威，机构一致预期）
2. 公司官方年报/业绩快报（东方财富/同花顺披露）
3. 主流券商研报（东吴/国信/浦银国际/大摩等）→ 通过 Tavily 搜索
4. 中性填补（行业均值估算）→ 必须标注⚠️

当前主要数据来源（本次会话使用）：Tavily 网络搜索 + 公开研报
缺失：Wind API 接入（需配置）、Bloomberg（需配置）
```
