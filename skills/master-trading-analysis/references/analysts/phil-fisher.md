# Phil Fisher 分析框架

来源：`ai-hedge-fund/src/agents/phil_fisher.py`

**核心理念**：深度调研（"Scuttlebutt"）寻找长期成长型公司。强调管理质量、R&D 投入和利润率稳定性，愿意长期持有。

---

## 六维加权评分（满分10分）

| 维度 | 权重 | 说明 |
|---|---|---|
| 成长质量 | 30% | 营收/利润增速及质量 |
| 利润率稳定 | 25% | 毛利率和净利率的稳定与趋势 |
| 管理效率 | 20% | ROIC、资产周转率、股权回报 |
| 估值 | 15% | P/E、P/FCF |
| 内部人活动 | 5% | 内部人净买卖 |
| 情绪 | 5% | 新闻情绪 |

---

## 综合信号规则

```
weighted_score = Σ (维度得分/10 × 维度权重) × 10

≥ 7.5 → bullish
≤ 4.5 → bearish
其他   → neutral
```

---

## Fisher 式 LLM Prompt

```
你是 Phil Fisher AI 代理：
1. 强调长期持有（5-10年视野），不关注短期波动
2. 管理层质量是最重要的非量化因素（可信赖 + 善于资本配置 + 关注员工）
3. 寻找 R&D 和创新持续投入的公司（未来增长护城河）
4. 利润率的稳定性和提升趋势是护城河的直接证据
5. 用"Scuttlebutt"方法：询问供应商、客户、竞争对手来验证公司护城河

推理时引用：
- ROIC 多期趋势
- 利润率变化轨迹
- 管理层历史资本配置记录
- 公司是否有新产品线/市场扩张计划
```

---

## 关键指标速查

```
核心：ROIC（>15%持续）, 净利率（>10%稳定）, 营收增速（>8%）
管理：ROE, 资产周转率趋势, 股份回购记录
估值：P/E（<30 可接受）, P/FCF（<25）
调研问题：护城河来源？R&D投入？管理层记录？
```
