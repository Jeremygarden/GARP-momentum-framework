# Bill Ackman 分析框架

来源：`ai-hedge-fund/src/agents/bill_ackman.py`

**核心理念**：激进投资者——寻找高质量企业中被市场低估的价值，并通过推动管理层改革（激进主义）来解锁潜力。集中持有少数高确信度标的。

---

## 四维评分体系（各满分5分，总分20分）

### 1. 业务质量（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 毛利率 | > 40% | +2（强品牌/定价权）|
| 毛利率 | > 20% | +1 |
| 营业利润率趋势 | 持续改善 | +1 额外 |
| FCF 转化率（FCF/净利润）| > 80% | +1 |
| 营收增速 | > 8% | +1 |

*毛利率是品牌护城河的最直接体现*

### 2. 资产负债表（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| D/E | < 2.0 | +1 |
| D/E | < 1.0 | +2 |
| 利息覆盖率 | > 3× | +1 |
| 利息覆盖率 | > 5× | +2 |
| 流动比率 | > 1.5 | +1 |
| 历史分红 | 连续发放 | +1 额外 |
| 股份数量趋势 | 减少（净回购）| +1 额外 |

### 3. 激进主义潜力（满分5分）

Ackman 寻找可以通过干预管理层来解锁价值的公司：

| 指标 | 条件 | 分值 |
|---|---|---|
| 净利率低于同业 > 20% | 是（改善空间大）| +3 |
| 营业利润率低于行业 | 是 | +2 |
| 资产回报率（ROA）< 5% | 是（资产利用不足）| +2 额外 |
| 管理层持股偏低（< 5%）| 是（利益不一致）| +1 额外 |

### 4. 估值与安全边际（满分5分）

使用 FCF 折现估算内在价值：

```
normalized_fcf = 近3年 FCF 均值
terminal_growth = 3%（保守永续增长）
discount_rate = 10%（Ackman 要求的最低回报）
intrinsic_value = normalized_fcf / (discount_rate - terminal_growth) = FCF × 14.3×
```

| 安全边际 | 条件 | 分值 |
|---|---|---|
| (内在价值 - 市值) / 市值 | > 30% | +5 |
| | > 15% | +3 |
| | > 0% | +1 |
| | ≤ 0% | 0 |

---

## 综合信号规则

```
总分 = 业务质量 + 资产负债表 + 激进主义潜力 + 估值，满分约20-22

≥ 70% × 20 = ≥ 14 → bullish
≤ 30% × 20 = ≤ 6  → bearish
其他               → neutral
```

## Ackman 式 LLM Prompt

```
你是 Bill Ackman AI 代理：
1. 寻找拥有强品牌/护城河但被低效管理拖累的高质量企业
2. 深入分析资本配置——管理层是否有效使用股东资金？
3. 识别激进主义机会：削减成本/改善管理层/战略重组能解锁多少价值？
4. 高度集中，每个持仓都必须是最高确信度
5. 估值框架：FCF 打折 10%，内在价值有 30% 以上安全边际才入场

推理风格：
- "这家公司拥有出色的品牌，但运营效率仅达到行业70%，若改善至行业均值，利润可提升 X%..."
- 引用具体财务数据支撑论点
- 明确指出潜在催化剂
```

---

## 关键指标速查

```
核心：毛利率（>40% 优）, FCF_yield, D/E（<2）, 利息覆盖（>3×）
激进主义：净利率 vs 同业, ROA, 管理层持股比例
估值：内在价值 = FCF × 14.3×（按10%折现率）
      安全边际 = (内在价值 - 市值) / 市值（>30% 才考虑）
```
