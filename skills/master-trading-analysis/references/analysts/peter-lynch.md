# Peter Lynch 分析框架

来源：`ai-hedge-fund/src/agents/peter_lynch.py`

**核心理念**：PEG 是关键指标，寻找被机构忽视的成长股，以日常生活观察发现"十倍股"。

---

## 评分体系（满分10分）

### 1. PEG 与 PE 评分（满分5分）

```
PEG = P/E ÷ EPS增速(%)
（EPS增速为小数时需 × 100 转为百分比）
```

| P/E | 分值 |
|---|---|
| < 15 | +2 |
| 15-25 | +1 |
| ≥ 25 | 0 |

| PEG | 分值 |
|---|---|
| < 1.0 | +3（极佳）|
| 1.0-2.0 | +2（良好）|
| 2.0-3.0 | +1（尚可）|
| ≥ 3.0 | 0 |

### 2. 盈利能力（满分4分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 营业利润率 | > 20% | +2 |
| 营业利润率 | 10%-20% | +1 |
| FCF | > 0 | +2 |

### 3. 成长稳定性（满分4分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 营收增速 | > 20% | +2 |
| 营收增速 | 10%-20% | +1 |
| EPS 增速 | > 20% | +2 |
| EPS 增速 | 10%-20% | +1 |

---

## 综合信号规则（满分13分）

```
≥ 7.5 分 → bullish
≤ 4.5 分 → bearish
其他      → neutral

信心度 = 得分 / 13 × 100
```

---

## Lynch 式 LLM Prompt

```
你是 Peter Lynch AI 代理：
1. PEG 是你最重要的估值工具——PEG < 1 是黄金机会
2. 喜欢"平凡但出色"的公司：产品简单、市场占有率逐步提升
3. 关注"十倍股"潜力：快速扩张阶段（开店/进入新市场/新产品线）
4. 对高负债和时髦概念保持怀疑
5. 若我的孩子都在用这个产品，那是个好信号

推理语言：
- "这家公司的 PEG 只有 X，这意味着..."
- "如果他们继续以这个速度扩张..."
- "用朴素的语言解释这门生意的增长逻辑"
```

---

## 关键指标速查

```
核心：PEG（< 1 为强信号），P/E（< 15 优），EPS增速
成长：营收增速（>20%优）、营业利润率（>20%优）
健康：FCF > 0，D/E 不宜过高
```
