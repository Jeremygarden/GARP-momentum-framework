# Stanley Druckenmiller 分析框架

来源：`ai-hedge-fund/src/agents/stanley_druckenmiller.py`

**核心理念**：宏观驱动的动量投资——在趋势确立时重仓出击，发现风险回报不对称的机会，快速止损。"当机会来临时，一定要下大注。"

---

## 四维评分体系

### 1. 成长动量（满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 营收增速 | > 25% | +4 |
| 营收增速 | > 15% | +3 |
| 营收增速 | > 8% | +2 |
| 营收加速（近期 > 历史均值）| 是 | +2 额外 |
| EPS 增速 | > 25% | +3 |
| EPS 增速 | > 15% | +2 |
| 利润率扩张（营业利润率提升）| 是 | +1 |

### 2. 市场动量（满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 1个月股价收益率 | > 5% | +2 |
| 3个月股价收益率 | > 15% | +3 |
| 6个月股价收益率 | > 25% | +3 |
| 12个月收益率 | > 30% | +2 |
| 价格在 52 周高点附近（> 90%）| 是 | +2 额外（强势突破）|

### 3. 估值（Druckenmiller 允许高估值，但有下限）

| 指标 | 条件 | 分值 |
|---|---|---|
| P/FCF | < 15 | +2 |
| P/FCF | < 25 | +1 |
| EV/EBIT | < 15 | +2 |
| EV/EBIT | < 25 | +1 |

*注：Druckenmiller 愿意为高增长支付高估值，但仍关注价格合理性*

### 4. 情绪与内部信号（辅助）

| 条件 | 分值 |
|---|---|
| 内部人净买入 | +2 |
| 机构持股增加 | +1 |
| 新闻情绪偏正面 | +1 |

## 综合信号规则

```
得分比率 ≥ 65% → bullish（高增长 + 动量确立）
得分比率 ≤ 35% → bearish（增速下滑 + 动量反转）
其他           → neutral
```

---

## 风险回报评估（Druckenmiller 最重视）

```
上行潜力：
  bullish case = 当前价 × (1 + EPS_growth × 5)^0.5
  upside = (bullish_case - current_price) / current_price

下行风险：
  bearish case = 当前价 × 0.70（假设 30% 最大回撤）
  downside = (current_price - bearish_case) / current_price

风险回报比 = upside / downside
  > 3:1 → 绝佳入场时机
  > 2:1 → 可以接受
  < 1:1 → 风险回报不对称，不参与
```

## Druckenmiller 式 LLM Prompt

```
你是 Stanley Druckenmiller AI 代理：
1. 首要问题：流动性和宏观环境是否有利？（利率/货币政策/市场情绪）
2. 寻找增长加速的公司——从15%到30%的加速比从30%持续到30%更有价值
3. 动量确认：股价趋势是否支持基本面？（两者共振才是最强信号）
4. 非对称风险回报：上涨空间 / 下行风险 > 3:1 才出手
5. 快速止损：如果论点失效，立即离场，不恋战

推理重点：
- 增长加速的来源（新产品/新市场/行业顺风）
- 宏观背景（加息/降息/汇率对业务的影响）
- 动量持续性（是否在趋势早期还是晚期）
- 明确指出停止持有的条件（thesis breaker）
```

---

## 关键指标速查

```
核心：营收增速（加速趋势）, 3-6个月价格动量, 风险回报比（>2:1）
宏观：利率环境, 行业顺风/逆风
估值：P/FCF（<25 可接受）, EV/EBIT（<25）
止损：论点核心假设若不成立，立即退出
```
