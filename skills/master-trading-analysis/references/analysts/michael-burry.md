# Michael Burry 分析框架

来源：`ai-hedge-fund/src/agents/michael_burry.py`

**核心理念**：逆向投资者，深度挖掘市场错误定价。用 FCF 收益率和 EV/EBIT 发现真正被低估的股票，并在市场过度悲观时布局。

---

## 四维评分体系

### 1. 价值分析（满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| FCF 收益率 | > 15% | +5 |
| FCF 收益率 | > 8% | +3 |
| FCF 收益率 | > 4% | +1 |
| EV/EBIT | < 8 | +3（极便宜）|
| EV/EBIT | < 12 | +2 |
| EV/EBIT | < 15 | +1 |
| EV/EBIT | ≥ 15 | 0 |
| P/FCF | < 10 | +2 |
| P/FCF | < 15 | +1 |

```
FCF 收益率 = FCF / 市值
EV = 市值 + 总债务 - 现金及等价物
```

### 2. 资产负债表强度（满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| D/E | < 0.3 | +4（极低杠杆）|
| D/E | < 0.7 | +2 |
| D/E | < 1.0 | +1 |
| 流动比率 | > 2.0 | +3 |
| 流动比率 | > 1.5 | +2 |
| 流动比率 | > 1.0 | +1 |
| 利息覆盖率 | > 10× | +3 |
| 利息覆盖率 | > 5× | +2 |
| 利息覆盖率 | > 2× | +1 |

### 3. 内部人交易信号（满分5分）

| 条件 | 分值 |
|---|---|
| 净内部购买 > 100,000 股 | +5（强烈买入信号）|
| 净内部购买 > 0 | +2 |
| 净内部购买 ≤ 0 | 0 |

```
净内部购买 = 近12个月所有内部人交易股数之和
（正 = 净买入，负 = 净卖出）
```

### 4. 逆向情绪分析（满分5分）

基于近期公司新闻情绪：

| 条件 | 分值 |
|---|---|
| 新闻负面占比 > 70%（市场极度悲观）| +5（逆向买入机会）|
| 新闻负面占比 > 50% | +3 |
| 新闻负面占比 > 30% | +1 |
| 新闻正面占比 > 70%（市场极度乐观）| -2（警惕过热）|

---

## 综合信号规则

```
总分 = 价值分 + 资产负债分 + 内部人分 + 逆向情绪分，满分约30

≥ 70% 满分 → bullish（FCF yield 高 + 低杠杆 + 内部人买入）
≤ 30% 满分 → bearish
其他        → neutral
```

---

## Burry 式 LLM Prompt

```
你是 Michael Burry AI 代理：
1. 用 FCF 收益率和 EV/EBIT 衡量真实价值（不用 P/E，利润可以作假）
2. 资产负债表强度至关重要——高杠杆公司直接排除
3. 内部人净买入是重要的正向信号（他们比我们更了解自己的公司）
4. 在市场恐慌、新闻悲观时寻找买入机会（逆向思维）
5. 短而有力的推理，引用具体数字

示例推理风格：
"FCF yield 12.8%. EV/EBIT 6.2. Debt-to-equity 0.4. Net insider buying 25k shares. Strong buy."
"FCF yield only 2.1%. Debt-to-equity 2.3. Management diluting shareholders. Pass."
```

---

## 关键指标速查

```
核心：FCF_yield（>8% 理想）, EV/EBIT（<12 理想）, D/E（<0.7）
逆向：新闻负面率, 净内部购买股数
公式：EV = market_cap + total_debt - cash
      FCF_yield = FCF / market_cap
```
