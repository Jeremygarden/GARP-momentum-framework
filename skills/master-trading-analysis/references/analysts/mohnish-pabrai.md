# Mohnish Pabrai 分析框架

来源：`ai-hedge-fund/src/agents/mohnish_pabrai.py`

**核心理念**："Dhandho"投资哲学——低风险、高回报，寻找安全边际极大的低风险翻倍机会（"Heads I win, Tails I don't lose much"）。

---

## 三维加权评分

### 1. 下行保护分析（权重 45%，满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 净现金头寸（现金 - 债务）| > 0 | +3（关键保护）|
| 流动比率 | ≥ 2.0 | +2 |
| 流动比率 | ≥ 1.2 | +1 |
| D/E | < 0.3 | +2（极低杠杆）|
| D/E | < 0.7 | +1 |
| FCF 连续正值（近3年均值 > 近更早3年均值）| 是 | +2（稳定增长）|
| FCF 近3年均为正 | 是 | +1 |

**关键原则**：下行保护是 Pabrai 最看重的，占总分权重 45%。

### 2. 现金收益率与估值（权重 35%，满分10分）

| 指标 | 条件 | 分值 |
|---|---|---|
| FCF 收益率 | > 10% | +4 |
| FCF 收益率 | > 7% | +3 |
| FCF 收益率 | > 4% | +2 |
| EV/EBIT | < 8 | +3 |
| EV/EBIT | < 12 | +2 |
| EV/EBIT | < 15 | +1 |
| P/FCF | < 12 | +3 |
| P/FCF | < 18 | +2 |

### 3. 翻倍潜力（权重 20%，满分10分）

基于5年自由现金流增长预测：

| 条件 | 分值 |
|---|---|
| 按增速5年后 FCF/当前市值 > 20%（极低估值）| +5 |
| 按增速5年后 FCF/当前市值 > 12% | +3 |
| 强复利迹象（再投资回报率 > 15%）| +3 额外 |
| 管理层持股比例 > 20% | +2 额外 |

---

## 综合信号规则

```
综合得分 = 下行分×0.45 + 估值分×0.35 + 翻倍分×0.20
（满分 = 10 × 0.45 + 10 × 0.35 + 10 × 0.20 = 10）

≥ 7.5 → bullish（低风险高回报，Pabrai 的理想标的）
≤ 4.0 → bearish
其他   → neutral
```

## Pabrai 式 LLM Prompt

```
你是 Mohnish Pabrai AI 代理：
1. Dhandho 哲学：寻找"零风险翻倍"机会——大赢/小输的不对称回报
2. 优先确保下行保护：净现金头寸 + 低杠杆 + 稳定 FCF
3. 只买便宜到吓人的股票（FCF收益率>8%，EV/EBIT<12）
4. 高度集中组合（5-10只，不分散）
5. 买入并长期等待，不做交易

推理结构：
- 翻倍情景："如果X，3年内市值可能..."
- 下行情景："即使最坏情况，损失不超过..."
- 关键数字：净现金/FCF收益率/EV-EBIT/管理层持股
```

