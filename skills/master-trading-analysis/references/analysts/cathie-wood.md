# Cathie Wood 分析框架

来源：`ai-hedge-fund/src/agents/cathie_wood.py`

**核心理念**：颠覆性创新是最大的投资机会。押注5年以上的指数级增长，愿意为高质量创新支付高估值，关注可寻址市场（TAM）的爆发。

---

## 三维评分体系（满分15分）

### 1. 颠覆性技术分析（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| R&D/营收比 | > 15% | +2（高投入创新）|
| R&D/营收比 | > 8% | +1 |
| 研发支出增速 | > 20% YoY | +2 |
| 研发支出增速 | > 10% YoY | +1 |
| 毛利率 | > 50%（SaaS/平台特征）| +1 |

### 2. 创新驱动增长（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 营收增速 | > 50% | +3（指数级增长）|
| 营收增速 | 25%-50% | +2 |
| 营收增速 | 15%-25% | +1 |
| 营收加速（近期增速 > 历史均值）| 是 | +2 额外 |

### 3. 高增长情景估值（满分5分）

*Wood 的估值不依赖当前盈利，而是5年后的市值预测*

| 条件 | 分值 |
|---|---|
| 按当前增速，5年后市值/现市值 > 5× | +3（潜在十倍股）|
| 5年后市值/现市值 2×-5× | +2 |
| 资本支出强度 > 10%（投入基础设施）且增速 > 20% | +2 额外 |
| 无分红（利润全部再投资）| +1 额外 |

```
5年预测市值 ≈ 当前营收 × (1 + growth_rate)^5 × 行业P/S中位数
```

---

## 综合信号规则

```
总分满分 15（原始）
≥ 70% × 15 = ≥ 10.5 → bullish
≤ 30% × 15 = ≤ 4.5  → bearish
其他                  → neutral
```

## Wood 式 LLM Prompt

```
你是 Cathie Wood AI 代理：
1. 只投资颠覆性技术/平台（AI、基因组、区块链、自动驾驶、能源存储、太空）
2. 以5年视角判断，忽略短期波动和当前盈利
3. 关注指数级增长曲线——从25%到40%再到100%的加速采用
4. 愿为真正颠覆性公司支付高估值（但需要 TAM 极大才能支撑）
5. 对"仅仅是渐进改进"的公司无兴趣

关键问题清单：
- 这家公司是否在创造新市场，而非瓜分已有市场？
- 边际成本是否趋近于零（软件/平台特征）？
- 网络效应是否存在？
- 5年后它会是一个"平台"还是一个"产品"？
```

---

## 关键指标速查

```
核心：营收增速（>25%优）, R&D/营收（>8%优）, 毛利率（>50%优）
增长：YoY加速趋势, 再投资率（低/零分红）
估值：5年预测市值/当前市值（TAM × 市场份额预测）
```
