# Growth Agent 分析框架

来源：`ai-hedge-fund/src/agents/growth_agent.py`

**核心理念**：纯量化成长因子分析——用线性回归捕捉营收/利润/FCF 的长期增长趋势斜率，识别真正处于成长轨道的公司。

---

## 四维增长趋势评分

*注：以下"趋势斜率"通过线性回归计算，正斜率=增长，负斜率=下滑*

### 1. 营收增长趋势（满分10分）

| 条件 | 分值 |
|---|---|
| 营收线性趋势斜率 > 0（持续增长）| +5 |
| 近2年营收 YoY > 15% | +3 |
| 近2年营收 YoY > 8% | +2 |
| 营收加速（最近一期增速 > 均值）| +2 额外 |

### 2. 盈利增长趋势（满分10分）

| 条件 | 分值 |
|---|---|
| EPS 线性趋势斜率 > 0 | +5 |
| 近期 EPS 增速 > 15% | +3 |
| 近期 EPS 增速 > 8% | +2 |
| EPS 加速（最近一期增速 > 历史均值）| +2 额外 |

### 3. FCF 增长趋势（满分10分）

| 条件 | 分值 |
|---|---|
| FCF 线性趋势斜率 > 0（持续正增长）| +5 |
| FCF 近3年均值 > 近更早3年均值（增长）| +3 |
| FCF 绝对值持续为正（≥5年）| +2 |

### 4. 内部人信号（满分10分，辅助）

| 条件 | 分值 |
|---|---|
| 近12个月内部人净买入 > 100,000 股 | +8 |
| 净买入 > 0 | +5 |
| 净卖出 | 0 |

---

## 综合信号规则

```
raw_score = (营收增长 + 盈利增长 + FCF增长 + 内部人信号) / 4

≥ 7.5 → bullish（多维成长共振）
≤ 4.5 → bearish（增长停滞）
其他   → neutral
```

## 趋势斜率计算方法

```python
# 简单线性回归斜率
def calculate_trend(data: list) -> float:
    """正值=向上趋势，负值=向下趋势，绝对值越大趋势越强"""
    n = len(data)
    x = list(range(n))
    sum_x = sum(x); sum_y = sum(data)
    sum_xy = sum(i*j for i,j in zip(x, data))
    sum_x2 = sum(i**2 for i in x)
    slope = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x**2)
    return slope
```

## 使用建议

**最适合场景**：
- 快速筛选成长股（无需大师视角，纯量化）
- 配合 Cathie Wood/Druckenmiller 分析验证成长质量
- 内部人信号作为最终确认

**局限性**：
- 不考虑估值（成长但昂贵的股票会获高分）
- 历史增长不代表未来（应结合护城河和行业分析）
- 对周期性行业有偏差（高成长可能只是周期高点）

---

## 关键指标速查

```
核心：营收/EPS/FCF 的线性趋势斜率（正负）
辅助：近期增速 vs 历史均值（加速/减速）
确认：内部人净买卖
避坑：周期股高峰期、成长减速的高估值股
```
