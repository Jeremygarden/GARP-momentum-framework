# Ben Graham 分析框架

来源：`ai-hedge-fund/src/agents/ben_graham.py`

**核心理念**：深度价值投资——买入被市场严重低估的资产，坚持安全边际，以内在价值为锚。

---

## 三大核心评分模块

### 1. 净流动资产价值（NCAV）

```
NCAV = 流动资产 - 总负债
NCAV/股 = NCAV / 流动股数
```

| 条件 | 分值 | 说明 |
|---|---|---|
| NCAV > 市值（净净股） | **+4** | Graham 经典强买信号 |
| NCAV/股 ≥ 当前股价 × 2/3 | +2 | 中度折价，部分安全边际 |
| NCAV/股 < 当前股价 × 2/3 | 0 | 不满足净净条件 |

### 2. 格雷厄姆数（Graham Number）

```
Graham Number = √(22.5 × EPS × 每股净资产BVPS)
（22.5 = 15×PE × 1.5×PB 的乘积上限）
```

| 条件 | 分值 |
|---|---|
| 格雷厄姆数 ≥ 当前价格 × 1.5 | +3（深度低估）|
| 格雷厄姆数 ≥ 当前价格 × 1.2 | +2 |
| 格雷厄姆数 ≥ 当前价格 | +1（轻微低估）|
| 格雷厄姆数 < 当前价格 | 0（高估）|

### 3. 财务稳健性

| 指标 | 条件 | 分值 |
|---|---|---|
| D/E | < 0.5 | +2 |
| 流动比率 | > 2.0 | +2 |
| 盈利稳定性 | 近7年无亏损 | +2 |
| 股息连续发放 | 近5年 | +1 |
| P/E | < 15 | +2 |
| P/B | < 1.5 | +1 |

---

## 综合信号规则

```
总分上限约为 13-17 分（取决于数据完整性）

得分比率 ≥ 70% → bullish
得分比率 ≤ 30% → bearish
其他          → neutral
```

---

## 格雷厄姆式 LLM Prompt

```
你是 Benjamin Graham AI 代理：
1. 坚持安全边际——只在价格远低于内在价值时买入（通常折价≥33%）
2. 优先使用格雷厄姆数和 NCAV 作为估值锚点
3. 强调资产负债表保守性（低杠杆、高流动性）
4. 要求盈利记录稳定（7年以上无亏损）
5. 不为成长支付高价，价格保护优先于增长预期

推理时引用：NCAV vs 市值、格雷厄姆数 vs 当前价格、安全边际百分比
```

---

## 关键指标速查

```
核心：NCAV, Graham Number = √(22.5 × EPS × BVPS)
安全边际 = (Graham_Number - Current_Price) / Current_Price
必查：D/E（<0.5）, 流动比率（>2）, P/E（<15）, P/B（<1.5）
历史：7年盈利记录, 5年股息记录
```
