# Warren Buffett 分析框架

来源：`ai-hedge-fund/src/agents/warren_buffett.py`

**核心理念**：以合理价格买入优秀企业，长期持有，要求护城河和可持续竞争优势。

---

## 评分体系（满分100）

### 1. 盈利能力（25分）
| 指标 | 条件 | 分值 |
|---|---|---|
| ROE | > 15% | +5 |
| 净利润率 | > 20% | +5 |
| 营业利润率 | > 15% | +5 |
| ROE 连续5年 > 15% | 是 | +10 |

### 2. 增长稳定性（25分）
| 指标 | 条件 | 分值 |
|---|---|---|
| 营收增速 | > 10% | +5 |
| EPS 增速 | > 10% | +5 |
| 营收连续增长（≥5年） | 是 | +10 |
| EPS 连续增长（≥5年） | 是 | +5 |

### 3. 财务稳健（25分）
| 指标 | 条件 | 分值 |
|---|---|---|
| D/E | < 0.5 | +10 |
| 流动比率 | > 1.5 | +5 |
| 利息覆盖率 | > 5× | +5 |
| 自由现金流为正 | 是 | +5 |

### 4. 估值合理性（25分）
| 指标 | 条件 | 分值 |
|---|---|---|
| P/E | < 25 | +10 |
| P/B | < 3 | +5 |
| FCF 收益率 | > 4% | +10 |

---

## 信号规则

| 综合得分 | 置信度 | 信号说明 |
|---|---|---|
| 90-100% | 高度 bullish | 优秀企业 + 合理估值，强买 |
| 70-89% | bullish | 好企业有护城河，估值尚可 |
| 50-69% | neutral | 混合信号，需更多信息或更低价格 |
| 30-49% | bearish | 超出能力圈或基本面存疑 |
| 10-29% | 高度 bearish | 差企业或严重高估 |

## 巴菲特式 LLM Prompt

```
你是 Warren Buffett AI 代理，依照其投资原则做决策：
1. 只买护城河宽、能长期维持竞争优势的企业
2. 坚持安全边际，不为增长支付过高价格
3. 偏好可预测现金流，讨厌复杂或难懂的业务
4. 长期持有，不频繁交易
5. 对管理层诚信度和资本配置能力有高要求

推理时请：
- 引用具体财务数据（ROE、FCF 收益率、D/E）
- 评估护城河（品牌/专利/网络效应/成本优势/转换成本）
- 用巴菲特的语言："我喜欢这门生意，因为...""这个价格买入让我感觉..."
- 明确给出 bullish/bearish/neutral 和置信度（0-100）
```

---

## 关键指标速查

```
必查：ROE（连续5年）, FCF, D/E, P/E, P/B, FCF_yield
护城河评估：毛利率趋势, 净利率稳定性, 营收可预测性
管理层：股份回购记录, 分红稳定性, 资本回报率（ROCE）
```
