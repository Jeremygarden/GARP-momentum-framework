# Charlie Munger 分析框架

来源：`ai-hedge-fund/src/agents/charlie_munger.py`

**核心理念**：以合理价格买入优质企业（而非以低价买入普通企业）。专注 ROIC、定价权、可预测性、保守管理层。

---

## 四维评分体系

### 1. 护城河质量（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| ROIC | > 15% 的期数 ≥ 80% | +3 |
| ROIC | > 15% 的期数 ≥ 50% | +2 |
| ROIC | > 15% 的期数 > 0 | +1 |
| 毛利率稳定性 | 近3年 std < 5% | +1 额外 |
| 毛利率趋势 | 逐期提升 | +1 额外 |

**护城河总分 = ROIC分 + 最高2分额外加成，上限5**

### 2. 管理层质量（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| D/E | < 0.7 | +1 |
| 现金/营收比 | 0.10 - 0.25 | +1（合理现金持有）|
| 内部购买比率 | > 50% 交易为买入 | +1 |
| 股份数量趋势 | 持续减少（回购）| +1 |
| 利息覆盖倍数 | > 5× | +1 |

### 3. 业务可预测性（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| 营收增速 std | < 5%（极稳定）| +3 |
| 营收增速 std | < 10%（较稳定）| +2 |
| 利润波动 | 近5年均正 | +1 |
| FCF 稳定 | 近5年均正 | +1 |

### 4. 估值合理性（满分5分）

| 指标 | 条件 | 分值 |
|---|---|---|
| FCF 收益率 | > 8% | +3 |
| FCF 收益率 | > 5% | +2 |
| FCF 收益率 | > 3% | +1 |
| 合理价值安全边际 | > 0% | +2 额外 |

*合理价值 = 归一化FCF / cost_of_equity（永续增长模型）*

---

## 综合信号规则

```
总分 = 护城河 + 管理层 + 可预测性 + 估值，满分约18-20
得分比率 = 总分 / 最大分

≥ 70% → bullish
≤ 30% → bearish
其他  → neutral
```

---

## 芒格式 LLM Prompt

```
你是 Charlie Munger AI 代理：
1. 寻找具有持久竞争优势的优质企业（护城河 = 定价权+ROIC长期>15%）
2. 关注管理层资本配置能力（低杠杆+回购+合理现金储备）
3. 要求业务高度可预测，讨厌复杂/周期性/需要频繁再投资的行业
4. 以合理估值买入，不贪图便宜货（"我宁可以合理价格买好公司"）
5. 集中投资少数高确信度标的

推理时引用：ROIC多期表现、FCF收益率、内部人买卖比、负债控制
```

---

## 关键指标速查

```
核心：ROIC（多期）, FCF_yield（>5%）, D/E（<0.7）
辅助：毛利率稳定性, 营收增速标准差, 内部人交易比
估值：合理价值 = 归一化FCF / 权益成本
```
