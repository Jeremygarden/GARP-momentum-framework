# 第一层：赛道景气筛选（坡度决定长度）

逻辑：成长股的根基是行业β。无赛道红利，个股很难走出独立行情。

---

## 1.1 基础门槛（硬过滤，任一不达标直接排除）

| 条件 | 标准 | 说明 |
|---|---|---|
| TAM（潜在市场规模） | **> 750亿人民币** | 原500亿上调，过滤极小众赛道 |
| 行业所处周期 | 导入期 / 成长期 | 排除成熟期、衰退期 |
| 行业未来3年预期CAGR | **> 15%** | 参考行业协会/产业链调研，非卖方预测 |

**TAM > 750亿的判断来源优先级：**
```
1. 行业协会/政府规划文件（最可信）
2. 产业链龙头年报中的市场空间描述
3. 第三方咨询机构（IDC/Gartner/中国信通院）
4. 卖方研究报告（参考，折扣使用）
```

---

## 1.2 赛道质量三维核实（强化修正，原版缺失）

通过基础门槛后，进行三维核实评分（满分30分），低于15分降级处理。

### 维度A：行业集中度（CR3）

CR3 = 前三家公司营收之和 / 行业总营收

| CR3水平 | 得分 | 逻辑 |
|---|---|---|
| > 60% | 10分 | 寡头格局，龙头定价权强，竞争有序 |
| 40%-60% | 7分 | 相对集中，有整合空间 |
| 20%-40% | 4分 | 分散竞争，价格战风险 |
| < 20% | 0分 | 高度分散，红海竞争，直接排除考虑 |

**实证参考：**
- 光伏组件2022年CR3约50%→快速下降至40%以下，伴随价格战开始，提前预警信号
- 白酒CR3稳定在60%+，茅台/五粮液护城河稳固
- 半导体设备（国产）CR3仅15-20%，竞争格局差，印证了高成长但难持续的特征

### 维度B：供给侧扩张速度

供给扩张比 = 行业产能/产量CAGR / 需求CAGR

| 供给扩张比 | 得分 | 逻辑 |
|---|---|---|
| < 0.8（供给跑不上需求）| 10分 | 卖方市场，价格有支撑 |
| 0.8-1.2（供需平衡）| 7分 | 健康状态 |
| 1.2-1.8（供给略超需求）| 3分 | 价格承压，需要高质量龙头才能转嫁 |
| > 1.8（产能严重过剩）| 0分 | 直接排除，价格战必然发生 |

**实证：**
- 光伏硅料2021年供给扩张比 > 2.5，2022年价格从均价300元/kg暴跌至60元/kg
- 储能电池2023-2024年供给扩张比 > 2，碳酸锂价格从60万/吨跌至8万/吨
- 电网设备2023年供给扩张比约0.9，价格稳定，印证赛道质量

### 维度C：价格趋势（行业产品/服务价格）

| 价格趋势 | 得分 | 说明 |
|---|---|---|
| 过去12个月价格上涨或稳定 | 10分 | 供需健康，定价权存在 |
| 价格小幅下降（<10%）| 6分 | 温和竞争，可接受 |
| 价格明显下降（10%-30%）| 2分 | 警惕信号，需龙头成本优势 |
| 价格暴跌（>30%）| 0分 | 直接排除，行业β已负 |

**数据来源：**
- 有色金属/能源：期货价格（LME、SHFE）
- 科技产品：IDC报告、公司ASP趋势
- 工业品：Wind行业价格指数、卓创资讯、mysteel

---

## 1.3 催化剂加分（+5分上限）

| 催化剂类型 | 加分 |
|---|---|
| 明确政策驱动（国家级规划/补贴）| +5 |
| 技术变革导致需求爆发（AI/替代旧技术）| +4 |
| 商业模式创新（订阅制/平台效应）| +3 |
| 仅有行业自身增长，无额外催化 | 0 |

---

## 1.4 强制排除项

无论其他得分多高，以下情况直接排除：

- 强周期行业顶部（价格处于过去3年历史高位90%分位以上）
- 技术路线正在被替代（如传统胶卷、功能机、LCD被OLED替代阶段）
- 政策明确收紧的行业（如游戏版号收紧期、教培双减）
- 供给扩张比 > 1.8 的产能过剩行业

---

## 赛道综合得分汇总

```
基础门槛（TAM/周期/CAGR）：通过/不通过（一票否决）

赛道质量得分（满分30分）：
  CR3得分（10）+ 供给扩张比得分（10）+ 价格趋势得分（10）
  + 催化剂加分（0-5）
  = 最高35分，标准化为0-100分

标准化公式：赛道质量评分 = (原始得分 / 35) × 100

阈值：
  赛道质量评分 < 43（原始 < 15分）→ 降级为白银级观察，不进核心池
  赛道质量评分 ≥ 43 → 进入第二层质量筛选
```
