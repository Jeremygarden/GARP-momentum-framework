# 第三层：估值安全边际（价格决定赔率）

核心改进：G折扣系数修正券商预期系统性高估，引入多维估值锚。

---

## 3.1 G（增速）的修正计算

**问题**：A股券商预期存在系统性乐观偏差（科技/成长板块平均高估20-40%）。

**修正公式：**

```
G_adjusted = min(
    历史3年净利润CAGR × 0.70,    # 历史折扣：防止"过去映射未来"
    券商一致预期CAGR × 0.80,      # 预期折扣：修正系统性乐观
    所在行业景气CAGR              # 行业上限：个股不可能长期跑赢行业2倍以上
)

注：若三个值均不可用，取：
    G_adjusted = 营收3年CAGR × 0.65
```

**数据来源优先级：**
1. Wind一致预期（≥3家覆盖券商的平均值）
2. Bloomberg consensus
3. 公司指引（管理层Guidance）× 0.85
4. 行业协会预测数据

**分行业折扣参考（基于历史高估程度调整）：**

| 行业 | 历史高估程度 | 建议额外折扣 |
|---|---|---|
| 半导体/AI硬件 | 严重（30-50%） | 预期 × 0.75（比默认更严格）|
| 光伏/新能源制造 | 严重（20-40%） | 预期 × 0.75 |
| 医药/CXO | 严重（20-35%） | 预期 × 0.78 |
| 消费品 | 轻度（5-15%） | 预期 × 0.88（略放宽）|
| 电网/基建 | 中度（10-20%） | 预期 × 0.82 |
| 银行/金融 | 轻度（订单可见）| 预期 × 0.90 |

## 3.2 PEG估值评分

使用 **G_adjusted** 计算修正PEG：

```
PEG_adjusted = 当前PE(TTM) / G_adjusted(%)
```

**评分细则：**

| PEG_adjusted | 基础得分 | 说明 |
|---|---|---|
| < 0.8 | **100分** | 黄金买点，显著低估 |
| 0.8 - 1.0 | **90分** | 合理偏低，优质标的首选区间 |
| 1.0 - 1.2 | **75分** | 合理，需质量维度接近满分 |
| 1.2 - 1.5 | **60分** | 偏贵，仅顶级赛道龙头可接受 |
| 1.5 - 2.0 | **40分** | 较贵，需极强催化剂 |
| > 2.0 | **20分** | 泡沫区，除非全面牛市模式 |

**顶级赛道龙头溢价规则：**
```
若满足以下所有条件，PEG上限可放宽至1.5：
  ① 赛道质量评分 > 80分（CR3 > 50%，供给无过剩）
  ② 质量评分 > 85分（ROE/ROIC/CFO均优秀）
  ③ 市占率行业第一且持续提升
  ④ 当前市场状态为中性或牛市
```

---

## 3.3 绝对估值锚（PE上限）

无论PEG多低，PE(TTM)超过以下上限，直接降级处理：

| 行业类型 | PE上限 | 逻辑 |
|---|---|---|
| 消费品蓝筹（茅台类）| 45倍 | 高确定性给溢价，但有上限 |
| 科技/AI成长 | 60倍 | 高增速支撑，但须有盈利 |
| 半导体/硬件 | 50倍 | 周期波动需保留安全边际 |
| 银行/金融 | 15倍（PB < 1.2）| 用PB辅助，PE意义有限 |
| 能源/资源 | 20倍（PB < 2）| 强周期，低PE才有安全边际 |
| 医药/CXO | 35倍 | 中等成长，不宜过高 |
| 工业/制造 | 30倍 | 成熟行业不给高溢价 |
| 亏损成长股 | 不适用（用PS替代）| PS < 10×（科技SaaS可至15×）|

**超过PE上限的处理：**
- 超过10%以内：估值得分×0.8
- 超过10-30%：估值得分×0.5，且不进入钻石/黄金梯队
- 超过30%：直接降至白银级或剔除

---

## 3.4 补充估值指标（加减分）

| 指标 | 规则 |
|---|---|
| EV/EBITDA vs 历史分位 | < 25%分位=+8分，25-50%=+3分，> 75%=-5分 |
| PB vs 历史分位（金融/重资产）| < 25%分位=+8分 |
| FCF Yield（自由现金流/市值）| > 5%=+8分，3-5%=+4分，< 2%=-5分 |
| 股息率（成熟行业加分）| > 3%=+5分，> 5%=+10分 |

---

## 估值综合得分

```
估值得分 = PEG评分（60%权重）+ PE上限检查（20%权重）+ 补充指标（20%权重）

例如：
  PEG_adjusted = 0.95 → PEG评分 = 90分
  PE(TTM) = 28倍（科技行业，上限60倍）→ 正常
  FCF Yield = 4% → +4分
  EV/EBITDA在历史30%分位 → +3分

  估值得分 = 90×60% + 100×20% + (100+4+3)/107×20% = 54+20+19.4 ≈ 93分
```
