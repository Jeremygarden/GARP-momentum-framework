# 执行SOP（标准作业流程）

> **核心架构原则**（不可随意修改）
> 1. **风险因子（R）全程固定 10%**，不参与动态调整，不参与线性插值
> 2. 风险因子承担双重角色：①常规加减分 ②一票否决权（触发即归零，与权重无关）
> 3. **PEG阈值和股息率加分参数随市场状态切换即时生效**（无缓冲），权重切换有5日缓冲
> 4. ERP < 0 时无论其他信号，立即触发全面防御，暂停新建仓

---

## 步骤0：市场状态判断（每周一，执行一次）

**数据获取（L3 tavily_extract 为主; 如果失败报错以mx-data skill为辅）：**

| 指标 | 数据源 |
|------|--------|
| 全A PE(TTM) 当前值 | 东方财富全A估值页 |
| 全A PE 10年历史百分位 | 手动记录表 / Wind |
| 北向10日累计净流入 | 东方财富北向资金页 |
| 上证指数20日历史波动率 | thsdk / 同花顺 |
| 10年国债收益率 | 中债官网 |
| ERP = 1/PE(全A) − 10年国债收益率 | 计算得出 |

**四维投票判断 → 确认市场状态（见 `market-regime.md`）：**

```
≥3个变量指向同一状态 → 确认切换
2:2分裂 → 以 PE百分位 + ERP 为主裁
```

**关键规则：**
```
□ ERP < 0 → 立即触发全面防御，暂停新建仓，持仓只减不增
□ 状态切换：权重切换有5日缓冲（线性过渡），PEG阈值/股息率参数即时切换
□ 状态切换事件写入 Bitable 日志（字段：日期/前状态/后状态/触发指标值）
□ 全A PE历史数据缺失时，使用沪深300 PE TTM代替
```

## 步骤1：数据获取链路（五级优先，按序执行，成功即止）

```
L1 thsdk / baostock（本地接口）
   → 量价、ROE、EPS、净利润、营收、经营现金流

L2 mx-data skill（行情数据 / 资金流向，不覆盖一致预期）

L3 tavily_extract（一致预期主链路）
   东方财富：https://data.eastmoney.com/report/profitforecast.jshtml?code={code}
   同花顺F10：https://basic.10jqka.com.cn/{code}/worth.html
   → 券商一致预期 EPS / PE / 净利润增速

L4 tavily_search 研报摘要（补充验证）
   搜索词："{股票名} {代码} 2025年报 净利润 ROE 盈利预测 2026 一致预期 EPS PE"

L5 中性填补（最后兜底）
   → 必须标注 ⚠️，记入数据缺口表，置信度标记为 ⭐⭐⭐ 以下
```

**数据刷新频率（分层策略）：**

| 因子 | 刷新频率 | 数据源 | 说明 |
|------|---------|--------|------|
| **M 动量** | **每周一**（每次执行）| L1 thsdk 实时量价 | 价格动量每周必须刷新 |
| **G/Q/V 基本面** | **2.1 / 5.1 / 8.1 / 11.1** | L1-L3 完整拉取 | 与季报披露对齐 |
| **一致预期数据** | **2.1 / 4.30 / 7.15 / 10.15** | L3-L4 研报 | 取最近 90-150天有效均值，超期数据降级⚠️ |

> 非刷新周期内，G/Q/V 使用上次缓存值，仅刷新 M，总分据此重新计算排名。

## 步骤2：初筛（每季度或重大行情后，广度过滤）

```
基础过滤条件（全部满足）：
  □ 行业TAM > 750亿，未来3年CAGR > 15%
  □ 资产负债率 < 70%（非金融）
  □ 商誉/净资产 < 35%
  □ 控股股东质押比 < 60%
  □ 近3年经营现金流非连续为负

预期：500-2000家 → 50-150家候选
```

---

## 步骤3：赛道质量复选（深度，50-150家 → 20-50家）

```
按行业分组，对每家公司评估：
  1. CR3（行业集中度）
  2. 供给扩张比（产能CAGR vs 需求CAGR）
  3. 行业产品价格趋势（过去12个月）

赛道质量评分 < 43分 → 降级或剔除
```

## 步骤4：五因子评分（精度，20-50家逐一打分）

**⚠️ 权重加载规则（必读）：**
```
W1-W4（G/Q/V/M）从 market-regime.md 按当前状态加载，合计 90%
W5（风险R）始终固定 10%，不随状态变化
```

**评分计算：**
```
[成长 G]
  G_adjusted = min(历史3年CAGR×0.70, 券商预期×0.80, 行业CAGR)
  成长总分 = 赛道质量评分 × 0.4 + G_adjusted得分 × 0.6

[质量 Q]
  按行业分组选择对应质量基准（layer2-quality.md）
  股息率附加分：按当前市场状态加载加减分参数（market-regime.md §6.2）

[估值 V]
  PEG_adjusted = PE(TTM) / G_adjusted(%)
  PEG评分：按当前市场状态加载 PEG阈值（market-regime.md §6.1）
  超过当前状态可接受上限 → V分 ≤ 40

[动量 M]
  价格动量（6M超额收益）× 55% + 盈利修正动量（3M预期变化）× 45%

[风险 R，固定10%]
  一票否决检查（触发即总分归零，与权重无关）：
    □ 控股股东质押 > 60%
    □ 近3年经营现金流连续为负
    □ 商誉/净资产 > 35%
    □ 财务造假迹象 / 退市风险警示
  常规加减分：±5-10分

  R 分含义：分越高 = 风险健康度越高（越安全）
    90-100：财务干净，无集中风险，治理优秀
    80-89 ：有1-2个需关注点，不触发否决，但需持续监控
    60-79 ：中等风险，减仓预警区，优先核查核心论点
    < 60  ：高风险，考虑减仓；< 40 触发一票否决归零

  ⚠️ R < 90 强制输出"风险披露"规则：
    每次榜单输出时，对 R < 90 的标的必须额外列出以下四项：
    ① 风险等级标注（如 R=80 → ⚠️ 中低风险）
    ② 风险背后含义（具体是什么风险，1-2句说清楚）
    ③ 相关实证数据（来自最近一期财报/公告的具体数字）
    ④ 复查建议（下次复查重点看什么指标）

    示例（老铺黄金 R=80）：
      ⚠️ 风险等级：中低风险（R=80）
      风险含义：快速扩张期现金流承压，负债结构恶化
      实证数据：2025年末存货+292%，短期借款+348%，经营现金流约-53亿
      复查建议：下期关注经营现金流是否转正、短借/营收比是否收窄

总分 = G×W1 + Q×W2 + V×W3 + M×W4 + R×0.10
梯队：> 85分=钻石 / 75-84=黄金 / 60-74=白银 / <60=剔除
```

## 步骤5：组合构建与买入纪律

```
仓位规则：
  □ 单一个股上限：8%（牛市泡沫期降至5%）
  □ 单一赛道上限：25%
  □ 总股票池：10-15只

买入时机（按当前状态的PEG阈值判断，market-regime.md §6.1）：
  首选：PEG_adjusted ≤ 当前状态"最优门槛"
  次选：PEG ≤ 当前状态"可接受上限" + 技术面右侧企稳
  动量确认：盈利修正上调 + 价格动量 > 行业中位数

禁止：
  □ 追高（最近1个月涨幅 > 20%）
  □ 市场状态切换5日缓冲期内重仓新建
  □ ERP < 0 期间新建任何仓位
```

---

## 步骤6：持仓管理与退出规则

```
持续持有条件（全部满足）：
  □ 核心赛道景气 + 质量护城河论点未变
  □ 盈利修正未连续下调2个季度
  □ 价格动量未持续跑输行业中位数 > 15% 超3个月

主动减仓触发（满足任意一条）：
  □ 总分 < 60分（降至白银以下）
  □ 赛道质量触发重大降级（供给扩张比 > 1.8 / CR3快速下降）
  □ 一票否决条件触发 → 立即清仓
  □ 市场状态切换至"深度熊市" → 组合总仓位降至50%以下
  □ PE(TTM) 超过行业绝对上限30%
```

## 步骤7：Top15 榜单刷新与输出规则

> **定时执行：每周一 UTC 02:00（北京时间 10:00），范围：A股沪深300 + 港股恒生核心成分股**

### 7.1 排名变化标记规则

| 情况 | 标记方式 |
|------|---------|
| 排名上升 N 位 | 排名后加 `🔼+N`（如 `#3 🔼+2`）|
| 排名下降 N 位 | 排名后加 `🔽-N`（如 `#7 🔽-1`）|
| 排名无变化 | 不标记 |
| 新进入榜单（黑马）| 加 `🌟` 前缀（见 7.2）|

**标记适用范围：无论什么原因导致的名次变化（含黑马挤压），均做升降标记。**
唯一例外：被黑马挤压但**自身评分无任何变化**的标的，标注说明"因新晋黑马位移"但不加 🔼🔽 符号。

### 7.2 黑马定义与标注规则

**黑马定义：** 上次 Top15 榜单中不存在、本次新进入 Top15 的标的，无论是：
- 从未被跟踪的全新标的
- 曾经在榜但已跌出榜单、本次重新进入的标的

均算当月黑马，用 `🌟` 标记当前排名位置，并附：

```
🌟 [股票名]（代码）  第X名
入榜实证：[最近1个季报亮眼数据 / 重大事件，如"2026Q1净利润+X%超预期"]
触发因子：[GARP框架中的核心触发点，如"PEG降至0.6进入黄金区 / ROE突破20%"]
```

### 7.3 榜单输出格式（Markdown，发至飞书群）

```
📊 GARP成长动能 A/H Top15 榜单
📅 刷新时间：{YYYY-MM-DD} 08:00 CST
🔁 本次刷新：M动量全量更新 / G/Q/V [是/否]完整刷新
📌 市场状态：{状态名称} | 当前权重：G={x}% Q={x}% V={x}% M={x}% R=10%

| 排名 | 股票 | 代码 | 总分 | 梯队 | 变化 | 2026E增速 | ROE | PEG |
|:---:|------|------|:---:|:---:|:---:|:---:|:---:|:---:|
| #1 | 中际旭创 | 300308 | 96.6 | 💎 | 🔼+1 | +120% | 44% | 0.24 |
| #2 | 天孚通信 | 300394 | 91.5 | 💎 | — | +58% | 39% | 0.79 |
| 🌟#6 | [新黑马] | ... | ... | ... | NEW | ... | ... | ... |
...

🌟 当月黑马：[股票名]（代码）第X名
   入榜实证：...
   触发因子：...

⚠️ 数据缺口：[股票A] ROE⚠️ / [股票B] 增速⚠️（中性填补）
🔗 Bitable：{url}
```
