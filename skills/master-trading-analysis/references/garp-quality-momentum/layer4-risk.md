# 第四层：风险排雷（决定生存底线）

逻辑：防止财务造假、大股东损害利益、系统性爆雷。触发阈值为一票否决，不计入评分。

---

## 4.1 一票否决条件（触发任意一条，直接剔除）

| 风险类型 | 一票否决阈值 | 实证依据 |
|---|---|---|
| 资产负债率 | > 70%（非银行/地产/基建）| 高杠杆在利率上升/信用收缩时致命 |
| 商誉/净资产比 | > 35% | 商誉减值爆雷（天神娱乐/华谊兄弟等案例）|
| 控股股东质押比 | > 60% | 质押爆仓风险，强制平仓引发踩踏 |
| 经营现金流 | 连续3年为负 | 业务不造血，依赖融资续命 |
| 审计意见 | 非标准无保留意见 | 财务真实性存疑 |
| 近2年监管处罚 | 重大信披违规/内幕交易 | 公司治理风险极高 |
| 实控人/高管 | 近1年减持超过5%的持股 | 内部人对公司前景悲观 |

---

## 4.2 风险评分（-30 ~ +10分）

### 扣分项

| 风险因子 | 扣分 |
|---|---|
| 商誉/净资产 20%-35%（未触发否决）| -10分 |
| 控股股东质押 40%-60%（未触发否决）| -8分 |
| 资产负债率 60%-70%（未触发否决）| -8分 |
| 关联交易占比 > 20%（潜在利益输送）| -8分 |
| 前五大客户集中度 > 60%（客户依赖）| -5分 |
| 近1年有董监高离职异常 | -5分 |
| 解禁压力：未来3个月解禁股占流通盘 > 10% | -5分 |
| 外资持股比例 < 1%（国际资金回避）| -3分 |

### 加分项

| 正向风险因子 | 加分 |
|---|---|
| 股息率 > 2%（提供安全边际）| +5分 |
| 无有息负债（净现金公司）| +5分 |
| 机构持股比例 > 30% 且稳定 | +3分 |
| 无质押，实控人持续增持 | +3分 |
| 近1年通过股权激励绑定核心人才 | +2分 |

---

## 4.3 风险综合评分输出

```
风险修正分 = 基础分(0) + 加分项 - 扣分项，范围 [-30, +10]

触发一票否决项 → 直接剔除，不计入总分
无否决项 → 风险修正分纳入总分计算（权重10%）

例：
  商誉/净资产 = 25% → -10分
  股息率 = 2.5% → +5分
  无质押 → +3分
  风险修正分 = 0 - 10 + 5 + 3 = -2分
```
