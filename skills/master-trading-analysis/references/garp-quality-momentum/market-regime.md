# 市场状态判断与动态权重切换

逻辑：固定权重在极端行情中系统性失效。引入四维市场状态变量，动态切换因子权重，降低系统性Beta风险同时最大化Alpha捕获。

---

## 一、市场状态判断变量（四维指标体系）

### 变量1：全市场估值分位（PE Percentile）

```
指标：万得全A PE(TTM) 或 沪深300 PE(TTM) 的历史百分位（滚动10年）

计算：当前PE / 历史同期最高PE × 100（百分位）

阈值：
  < 30%分位 → 低估区（熊市/底部）
  30%-50%分位 → 合理偏低（震荡偏熊）
  50%-70%分位 → 合理区间（中性/震荡）
  70%-85%分位 → 偏贵区（震荡偏牛）
  > 85%分位 → 高估区（牛市/泡沫期）
```

### 变量2：北向资金连续流向（Foreign Flow Trend）

```
指标：沪深港通北向资金10日累计净流入

计算：过去10个交易日北向净流入之和（亿元人民币）

阈值：
  > +200亿 → 强烈净流入（外资看多，风险偏好高）
  +50亿 ~ +200亿 → 温和净流入
  -50亿 ~ +50亿 → 中性
  -200亿 ~ -50亿 → 温和净流出
  < -200亿 → 强烈净流出（外资恐慌，风险偏好低）
```

### 变量3：市场波动率状态（Volatility Regime）

```
指标：上证50 ETF期权隐含波动率（IV）或 VIX China 指数
      若无期权数据，使用上证指数20日历史波动率年化值

阈值：
  < 15%（年化）→ 低波动，市场平静（可能是牛市中段）
  15%-25% → 正常波动区间（中性）
  25%-35% → 偏高波动（震荡加剧，风险偏好下降）
  > 35% → 高度恐慌（熊市/危机时刻，但可能是底部信号）
```

### 变量4：股债收益差（Equity Risk Premium / ERP）

```
指标：股债收益差 = 万得全A股息率（或盈利收益率E/P）- 10年期国债收益率

计算：E/P(全A) = 1 / PE(全A TTM)
      ERP = E/P - 十年国债收益率(%)

历史均值约 3-4%（A股），标准差约1.5%

阈值：
  ERP > 均值+2σ（>6%）→ 股票极度便宜，强烈做多信号（底部区域）
  ERP > 均值+1σ（>4.5%）→ 股票偏便宜，看多偏好
  ERP 在均值附近（3%-4.5%）→ 中性估值
  ERP < 均值-1σ（<2%）→ 股票偏贵
  ERP < 均值-2σ（<0.5%）→ 股票极度贵，谨慎做多（顶部区域）
```

## 二、市场状态综合判断矩阵

结合四个变量，判断当前所处的市场状态：

| 状态名称 | PE分位 | 北向10日 | 波动率 | ERP | 综合信号 |
|---|---|---|---|---|---|
| **深度熊市/底部** | < 30% | < -200亿 | > 35% | > 6% | 极度悲观，反向布局时机 |
| **震荡偏熊** | 30%-50% | -200亿~+50亿 | 25%-35% | 4.5%-6% | 谨慎，防御为主 |
| **中性震荡** | 50%-70% | -50亿~+200亿 | 15%-25% | 3%-4.5% | 平衡配置，基本面主导 |
| **震荡偏牛** | 70%-85% | +50亿~+200亿 | 15%-20% | 2%-3% | 景气赛道进攻 |
| **牛市/泡沫期** | > 85% | > +200亿 | < 15%（低波牛）| < 2% | 追趋势，严控估值上限 |

**判断规则（四变量投票机制）：**
```
各变量独立判断所属状态，取多数票（≥3个变量指向同一状态）确认。
若变量信号冲突（2:2分裂），以"PE分位"和"ERP"的判断为主（估值维度权重更高）。
```

## 三、五因子动态权重矩阵

### 状态1：深度熊市 / 底部区域

**目标**：最大化安全边际，防御资本，等待反转信号。

| 因子 | 动态权重 | 变化方向 | 逻辑 |
|---|---|---|---|
| 成长（Growth）| **15%** | ↓ -10% | 熊市中高成长股估值杀伤最大 |
| 质量（Quality）| **45%** | ↑ +10% | 高质量公司抗跌，底部反弹更快 |
| 估值（Valuation）| **25%** | ↑ +10% | 便宜是最好的保护，低估值公司下行有限 |
| 动量（Momentum）| **5%** | ↓ -10% | 熊市中价格动量失效，下跌趋势不可追 |
| 风险修正（Risk）| **10%** | = | 维持不变，风险排雷始终关键 |

**特别规则：**
- ERP > 6%时，启动"逆向加仓模式"：降低动量要求（动量得分 > 40即可）
- 股息率 > 4%的个股自动加20分（熊市的股息安全垫）
- 关注破净（PB < 1）的高质量标的

---

### 状态2：震荡偏熊

**目标**：防御为主，精选质量最强标的，控制仓位。

| 因子 | 动态权重 | 变化方向 | 逻辑 |
|---|---|---|---|
| 成长（Growth）| **20%** | ↓ -5% | 成长溢价收窄，降低权重 |
| 质量（Quality）| **40%** | ↑ +5% | 质量因子提供下行保护 |
| 估值（Valuation）| **20%** | = | 维持不变 |
| 动量（Momentum）| **10%** | ↓ -5% | 动量可信度下降 |
| 风险修正（Risk）| **10%** | = | 维持 |

---

### 状态3：中性震荡（默认权重）

**目标**：五因子均衡配置，基本面主导选股。

| 因子 | 动态权重 | 说明 |
|---|---|---|
| 成长（Growth）| **25%** | 默认基准 |
| 质量（Quality）| **35%** | 默认基准 |
| 估值（Valuation）| **15%** | 默认基准 |
| 动量（Momentum）| **15%** | 默认基准 |
| 风险修正（Risk）| **10%** | 默认基准 |

---

### 状态4：震荡偏牛

**目标**：加大景气赛道的进攻力度，适度放宽估值要求。

| 因子 | 动态权重 | 变化方向 | 逻辑 |
|---|---|---|---|
| 成长（Growth）| **35%** | ↑ +10% | 景气赛道β主导，成长溢价扩大 |
| 质量（Quality）| **30%** | ↓ -5% | 质量要求略降，但不放弃 |
| 估值（Valuation）| **10%** | ↓ -5% | 市场给予更高估值容忍度 |
| 动量（Momentum）| **15%** | = | 维持，动量确认趋势 |
| 风险修正（Risk）| **10%** | = | 维持 |

**特别规则：**
- PEG上限从默认1.5放宽至1.8（顶级赛道龙头）
- PE绝对上限提高10%（科技成长股从60倍至65倍）
- 赛道质量评分权重提升：优先选择CR3高、供给无过剩的强势赛道

---

### 状态5：牛市 / 泡沫期

**目标**：追趋势捕获动量，严控组合集中度，随时准备快速减仓。

| 因子 | 动态权重 | 变化方向 | 逻辑 |
|---|---|---|---|
| 成长（Growth）| **35%** | ↑ +10% | 牛市中景气叙事主导 |
| 质量（Quality）| **25%** | ↓ -10% | 市场不分质量，但不能完全忽视 |
| 估值（Valuation）| **10%** | ↓ -5% | 估值在牛市中几乎失效 |
| 动量（Momentum）| **20%** | ↑ +5% | 追趋势，动量是主要选股信号 |
| 风险修正（Risk）| **10%** | = | 维持，但额外关注退出规则 |

**泡沫期特别风控规则：**
- 单只股票仓位上限从8%降至5%（防黑天鹅）
- 设置组合整体止盈线：若组合净值达到+50%（当年），触发20%仓位止盈
- 持续监控PE分位：若超过90%分位，无论其他信号如何，开始减仓

## 四、市场状态切换的缓冲机制

避免频繁切换造成交易摩擦：

```
切换规则：
1. 四个变量中≥3个指向新状态，且持续5个交易日以上，才触发权重切换
2. 切换不是瞬间完成，而是在5个交易日内线性过渡
3. 单季度内切换次数上限：3次（防止过度交易）

例：从"中性震荡"切换到"震荡偏牛"
Day 1-5: 仍使用中性权重（确认信号）
Day 6: 开始切换，每天调整1/5的权重差
Day 10: 完全切换到偏牛权重
```

## 五、动态权重汇总对照表

> ⚠️ **架构原则（2026-03-30 确认）**：
> - **风险因子（Risk）全程固定 10%，不参与动态调整，不参与线性插值过渡**
> - 风险因子承担双重角色：①常规加减分（±5-10分）②一票否决权（触发即归零，与权重无关）
> - 降低风险权重不影响否决功能，但会扭曲普通风险信号的影响力，故保持不变
> - 其余四个因子（G/Q/V/M）在 90% 的空间内线性插值过渡，风险因子始终占剩余 10%

| 因子 | 💀 深度熊市 | 📉 震荡偏熊 | ➡️ 中性（默认）| 📈 震荡偏牛 | 🚀 牛市/泡沫 |
|---|---|---|---|---|---|
| 成长 G | 15% | 20% | **25%** | 35% | 35% |
| 质量 Q | 45% | 40% | **35%** | 30% | 25% |
| 估值 V | 25% | 20% | **15%** | 10% | 10% |
| 动量 M | 5% | 10% | **15%** | 15% | 20% |
| **风险 R** | **10%** | **10%** | **10%** | **10%** | **10%** |
| **合计** | 100% | 100% | 100% | 100% | 100% |

**设计逻辑验证：**
- 熊市：质量(45%) + 估值(25%) = 70%，防御为主 ✅
- 偏牛：成长(35%) + 动量(15%) = 50%，进攻为主 ✅
- 全程：风险修正 **固定10%**，底线不随市场情绪漂移 ✅
- Alpha来源：质量因子在任何状态下权重最高（25%-45%），长期Alpha的核心来源 ✅

## 六、PEG 阈值与股息率参数随状态动态切换

> **架构说明**：PEG 和股息率属于**个股/行业层面估值变量**，不进入四维市场状态判断矩阵（避免与PE百分位维度重叠）。
> 但其**参数阈值**随市场状态联动切换，在估值因子（V层）和风险加减分中生效。

### 6.1 PEG 阈值随状态切换

| 市场状态 | PEG 最优门槛 | PEG 可接受上限 | 逻辑说明 |
|---------|:-----------:|:-----------:|---------|
| 💀 深度熊市 | < 0.5 | 0.7 | 熊市要求极度安全边际，PEG门槛收严 |
| 📉 震荡偏熊 | < 0.7 | 1.0 | 偏保守，只买明显低估标的 |
| ➡️ 中性震荡 | < 1.0 | 1.3 | 默认标准，PEG<1为优先 |
| 📈 震荡偏牛 | < 1.3 | 1.5 | 顶级赛道龙头可适度放宽 |
| 🚀 牛市/泡沫 | < 1.5 | 1.8 | 景气叙事主导，PEG上限放宽；超过1.8无论赛道一律降级 |

**评分映射（估值因子V，满分100）：**
```
各状态下 PEG ≤ 最优门槛  → 90-100分
各状态下 PEG 在门槛~可接受上限之间 → 60-89分（线性插值）
各状态下 PEG > 可接受上限 → 40分以下（不符合当前状态要求）
```

### 6.2 股息率参数随状态切换（加减分项，附加于 Q 质量因子）

> **为何附加于 Q 因子而非独立因子**：股息率反映企业现金回报能力与管理层资本分配质量，本质是质量信号而非成长信号。

| 市场状态 | 股息率加分规则 | 股息率减分规则 | 逻辑说明 |
|---------|-------------|-------------|---------|
| 💀 深度熊市 | > 4%：+15分；> 2.5%：+10分；> 1.5%：+5分 | < 0.5% 且非成长期：-5分 | 熊市股息是"安全气囊"，现金回报有价值 |
| 📉 震荡偏熊 | > 3%：+10分；> 1.5%：+5分 | < 0.5% 且非高成长：-3分 | 偏保守，高股息提供下行保护 |
| ➡️ 中性震荡 | > 3%：+5分 | 无额外减分 | 中性状态股息率作为辅助信号 |
| 📈 震荡偏牛 | > 4%（高成长公司）：+5分 | 无减分（成长期可以不分红）| 偏牛时股息率重要性下降 |
| 🚀 牛市/泡沫 | 不额外加分 | 不额外减分 | 泡沫期成长叙事主导，股息率信号失效 |

**特别说明：**
```
股息率 > 1.5% 在熊市/偏熊下是入选辅助条件，但不是一票否决条件
高成长期公司（净利润增速 > 30%）免除股息率减分惩罚
金融/能源等周期性高股息行业，股息率需结合"穿越周期ROE"判断，避免价值陷阱
```

### 6.3 牛熊状态 PEG + 股息率联合信号汇总

| 市场状态 | PEG 信号 | 股息率信号 | 综合含义 |
|---------|---------|---------|---------|
| 熊市 | 收严至 < 0.7 | 加重权重（> 1.5% 加分）| "便宜 + 有现金回报" = 熊市生存标准 |
| 偏熊 | < 1.0 | 适度加分 | "合理估值 + 一定保护" |
| 中性 | < 1.0（默认）| 辅助参考 | 标准 GARP 框架 |
| 偏牛 | 放宽至 < 1.5 | 基本忽略 | "景气赛道龙头，估值可容忍" |
| 牛市 | 放宽至 < 1.8 | 完全忽略 | "追趋势，但有绝对PEG上限防泡沫" |

## 七、缓冲机制完整规则（修订版）

```
切换规则：
1. ≥ 3个变量指向新状态，且持续 5 个交易日以上，触发权重切换
2. 切换采用线性过渡，5个交易日内平滑完成（每日调整 1/5 权重差）
3. 风险因子（R=10%）始终固定，不参与线性插值
4. PEG阈值和股息率加分规则随状态切换同步生效（无缓冲，立即生效）
5. 单季度内权重切换次数上限：3次
6. ERP < 0 时，无论其他信号，立即触发全面防御模式（暂停新建仓）

线性过渡示例（中性 → 偏牛，仅G/Q/V/M参与）：
  Day 1-5：确认信号，使用旧权重
  Day 6: G=27% Q=33% V=14% M=16% R=10%（固定）
  Day 7: G=29% Q=32% V=13% M=16% R=10%
  Day 8: G=31% Q=31% V=12% M=16% R=10%
  Day 9: G=33% Q=31% V=11% M=15% R=10%
  Day 10+: G=35% Q=30% V=10% M=15% R=10%（完成切换）
```
