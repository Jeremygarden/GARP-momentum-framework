# 第五层：动量确认（市场时机决定买点）

逻辑：正确的公司在错误的时机买入仍会亏损。动量因子确认市场对基本面的认同，降低价值陷阱风险。

---

## 5.1 价格动量（Price Momentum）

权重：动量因子总分的 **55%**

### 计算方式

```
价格动量得分 = 个股6个月收益率 vs 所在行业中位数6个月收益率的超额收益

超额收益 = R_stock(6M) - R_sector_median(6M)
```

### 评分细则

| 超额收益 | 得分 | 说明 |
|---|---|---|
| > +15% | 100分 | 强势领涨，市场高度认可 |
| +8% ~ +15% | 85分 | 明显跑赢，动量强 |
| +3% ~ +8% | 70分 | 适度领先，健康动量 |
| -3% ~ +3% | 55分 | 与行业同步，中性 |
| -8% ~ -3% | 35分 | 略跑输，需关注原因 |
| < -8% | 15分 | 明显跑输，可能是价值陷阱或行业问题 |

**关键说明：**
- 使用6个月而非12个月，因A股市场轮动快，12个月动量在A股的有效性弱于美股
- 排除最近1个月的超强上涨（防止追高）：若最近1个月涨幅 > 15%，动量得分×0.7
- 计算基于前复权价格

---

## 5.2 盈利修正动量（Earnings Revision Momentum）

权重：动量因子总分的 **45%**

逻辑：分析师预期上调比绝对预期值更有信号价值——这意味着基本面正在改善且市场尚未充分定价。

### 计算方式

```
盈利修正方向 = 过去3个月券商对当年/次年净利润预期的变化方向和幅度

修正幅度 = (当前一致预期 - 3个月前一致预期) / |3个月前一致预期|
```

### 评分细则

| 修正状态 | 得分 | 说明 |
|---|---|---|
| 上调 > +10% | 100分 | 强烈正向信号，基本面超预期 |
| 上调 +5% ~ +10% | 85分 | 明显改善 |
| 上调 +2% ~ +5% | 70分 | 轻度上调，温和正向 |
| 维持（-2% ~ +2%）| 55分 | 中性，无明显变化 |
| 下调 -2% ~ -5% | 35分 | 轻度下调，需关注 |
| 下调 -5% ~ -10% | 20分 | 明显下调，基本面恶化 |
| 下调 > -10% | 5分 | 严重下调，业绩可能暴雷 |

**无覆盖券商的处理（小市值/冷门股）：**
```
若覆盖券商 < 2家，盈利修正动量得分 = 55分（中性，不加分不减分）
若无覆盖，完全用价格动量替代：动量总分 = 价格动量得分
```

---

## 5.3 动量综合得分

```
动量总分 = 价格动量得分 × 55% + 盈利修正动量得分 × 45%

例：
  价格动量：个股6M +12% vs 行业中位数 +4% → 超额 +8% → 85分
  盈利修正：过去3个月预期上调 +7% → 85分
  动量总分 = 85 × 55% + 85 × 45% = 85分
```

---

## 5.4 动量的负向保护机制

以下情况触发动量保护，自动降低动量得分：

| 保护触发条件 | 处理方式 |
|---|---|
| 最近1个月股价涨幅 > 20%（追高风险）| 动量总分 × 0.65 |
| 最近1个月成交量较前3月均值扩张 > 3× | 动量总分 × 0.75（爆量追高）|
| 盈利修正下调 > -5% 但价格还在涨 | 动量总分 × 0.7（背离预警）|
| 行业已进入过热状态（赛道质量分<40）| 动量总分 × 0.6 |

---

## 5.5 动量因子的市场状态适配

动量因子本身也有失效场景，在 `market-regime.md` 中有针对性的权重切换：
- **牛市末期/泡沫期**：降低动量权重（趋势延伸但基本面已透支）
- **熊市/杀估值期**：提高动量权重（跌得少的才是真强）
- **震荡市**：动量维持标准权重，与基本面共同验证
