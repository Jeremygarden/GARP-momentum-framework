# 第二层：盈利质量筛选（护城河决定确定性）

逻辑：只有高质量的增长才给估值溢价。分行业差异化基准，解决统一ROE门槛的行业偏差问题。

---

## A股/H股核心行业分类与质量基准

### 行业分组 I：大金融（银行 / 保险 / 券商）

**为何不用ROE**：银行ROE受杠杆率影响，用ROE难以区分质量优劣；保险/券商受市场波动影响极大。

| 质量指标 | 银行 | 保险 | 券商 |
|---|---|---|---|
| 核心指标1 | ROA > 0.9%（头部可至1.2%）| 综合成本率 < 97% | ROE > 10%（行业特殊）|
| 核心指标2 | 净息差（NIM）> 1.8% | 新业务价值增速 > 10% | 资管/财富管理收入占比 > 30% |
| 核心指标3 | 不良贷款率 < 1.5% | 偿付能力充足率 > 150% | 杠杆率（自营/净资本）< 4× |
| 现金流替代 | 拨备覆盖率 > 200% | 内含价值增速 > 10% | — |
| 评分满分 | 100 | 100 | 100 |

**银行评分细则：**
```
ROA: > 1.2%=100, 1.0-1.2%=85, 0.9-1.0%=70, < 0.9%=40
NIM: > 2.0%=100, 1.8-2.0%=80, 1.5-1.8%=60, < 1.5%=30
不良率: < 1.0%=100, 1.0-1.5%=80, 1.5-2.0%=50, > 2.0%=20
拨备覆盖率: > 300%=100, 200-300%=80, 150-200%=60, < 150%=30
```

---

### 行业分组 II：重资产制造业（电力电网 / 钢铁铝 / 工程机械 / 建材）

**为何用ROIC而非ROE**：重资产行业ROE受负债结构影响，ROIC更真实反映资本回报效率。

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| ROIC | > 12%（扣税后）| > 18%=100分 |
| 经营现金流/净利润 | > 0.85 | > 1.1=100分 |
| 资产负债率 | < 55%（非金融）| < 40%=100分 |
| 固定资产周转率趋势 | 近3年稳定或上升 | 上升=加分 |
| 在手订单/年营收 | > 1.5×（可选）| > 2×=加分 |

**ROIC计算：**
```
ROIC = EBIT × (1 - 税率) / 投入资本
投入资本 = 股东权益 + 有息负债 - 超额现金
```

---

### 行业分组 III：消费品（白酒 / 食品饮料 / 零售 / 餐饮）

**特点**：轻资产，护城河来自品牌，ROE适用但需配合利润率稳定性。

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| ROE | > 18%（行业偏高）| > 25%=100分 |
| 经营现金流/净利润 | > 0.9 | > 1.1=100分 |
| 毛利率稳定性 | 近3年毛利率波动 < 3% | < 1%=100分 |
| 应收账款周转天数 | < 30天（快消）| < 15天=100分 |
| 存货周转天数 | < 90天（白酒例外可至3年）| 行业对比 |
| 净利率趋势 | 近3年维持或提升 | 持续提升=加分 |

**白酒特殊处理**：预收款/合同负债持续增长视为需求强劲的正向信号（茅台模式）。

---

### 行业分组 IV：科技/互联网（AI软件 / 云计算 / 纯互联网平台）

**特点**：早期可能亏损，ROE/ROIC意义有限，用Rule of 40 + FCF margin。

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| Rule of 40 | 营收增速 + 净利润率 > 40% | > 60=100分 |
| FCF Margin | 自由现金流/营收 > 15% | > 25%=100分 |
| 客户留存率/NRR | > 100%（SaaS）| > 120%=100分 |
| 毛利率 | > 50% | > 70%=100分 |
| 研发投入/营收 | > 10%（持续创新）| 10-20%=合理 |

**Rule of 40计算：**
```
= 营收YoY增速(%) + 调整后净利润率(%)
若亏损：净利润率为负数（如营收增速30%，净利润率-5%=25分，不达标）
```

---

### 行业分组 V：AI硬件 / 半导体 / 电子元器件

**特点**：周期性叠加成长性，需区分周期位置。

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| ROIC | > 12% | > 20%=100分 |
| 经营现金流/净利润 | > 0.8 | > 1.0=100分 |
| 库存周转天数 | < 90天（高于90天警惕过库存）| < 60天=100分 |
| 研发支出/营收 | > 8% | > 15%=100分 |
| 客户集中度 | 前五大客户收入占比 < 60% | < 40%=100分 |
| 毛利率趋势 | 近3年维持或上升 | 连续上升=加分 |

**客户集中度过高（>60%）直接降级**：单一大客户砍单风险（参考立讯精密/歌尔股份苹果依赖案例）。

---

### 行业分组 VI：能源（煤炭 / 石油 / 天然气 / 新能源发电）

**特点**：强周期，质量评估需看穿周期底部盈利能力。

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| 穿越周期ROE | 近5年平均ROE > 10% | > 15%=100分 |
| 现金成本/价格 | 成本分位处于行业前30%低成本 | 前10%=100分 |
| 净债务/EBITDA | < 2.0× | < 1.0×=100分 |
| 分红率 | > 40%（能源成熟期应返还现金）| > 60%=100分 |
| 储量替代率 | > 100%（油气专用）| > 120%=100分 |

---

### 行业分组 VII：金属 / 矿产（铜铝锂钴/黄金/铁矿）

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| 成本曲线位置 | 全球成本曲线前50%（低成本） | 前25%=100分 |
| 净债务/EBITDA | < 1.5× | < 0.5×=100分 |
| 资源储量年限 | > 10年 | > 20年=100分 |
| AISC（黄金全维持成本）| < 1200美元/盎司（黄金）| < 1000=100分 |
| 穿越周期FCF | 在商品价格-20%情景下FCF仍为正 | =加分 |

---

### 行业分组 VIII：医疗 / 医药 / 生物科技

| 质量指标 | 标准 | 满分标准 |
|---|---|---|
| ROE | > 15% | > 22%=100分 |
| 经营现金流/净利润 | > 0.85 | > 1.0=100分 |
| 研发管线丰富度 | 在研品种 > 3个处于二期以上临床 | > 5个=100分 |
| 专利到期风险 | 未来3年核心产品无专利悬崖 | 无风险=100分 |
| 净利率 | > 15% | > 25%=100分 |

---

### 行业分组 IX：房地产 / 建筑工程（谨慎参与）

当前周期（2024-2026年）建议整体回避，除非：
- 物业管理公司（轻资产，独立评估）
- 保障房/REITs标的
- 海外地产项目公司

若必须评估，关键指标：现金短债比 > 1.5、净负债率 < 60%、预收款/有息负债 > 1。

## 通用盈利质量指标（所有行业补充）

以下两项适用所有行业，作为加减分项：

| 指标 | 加减分规则 |
|---|---|
| 经营现金流/净利润 | > 1.0=+10分，0.8-1.0=0分，0.6-0.8=-10分，< 0.6=-20分 |
| Rule of 40（科技行业强制，其他行业参考）| > 60=+10分，40-60=0分，< 40=-5分 |
