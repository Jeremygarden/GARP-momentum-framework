# 第二层：板块强度（聚焦热点·拒绝孤狼）

**标的必须属于当日强势热点板块，拒绝独立走强的"孤狼股"。**

---

## 四项板块条件

### 1. 板块排名

| 要求 | 说明 |
|---|---|
| 标的所在板块涨幅排名 **前3** | 当日行业/概念板块涨幅榜 TOP3 |
| 若无明确主线，当日前3板块均不突出 | 缩减选股范围或空仓 |

**验证方式：**
- 东方财富板块排行：`quote.eastmoney.com` 行业板块涨幅
- thsdk：`ths.ths_industry()` / `ths.ths_concept()`

---

### 2. 板块涨停数量

| 要求 | 说明 |
|---|---|
| 板块内涨停个股 **≥ 2只** | 确保有龙头带动效应，非单兵突进 |
| 具备明确龙头标的 | 板块内有1只明显主导的强势龙头 |

```
板块内涨停数 = 0 或 1 → 板块缺乏合力，拒绝参与
板块内涨停数 ≥ 2，有龙头 → 板块具备赚钱效应
```

---

### 3. 板块指数走势

| 要求 | 说明 |
|---|---|
| 板块指数当日收红（正涨幅）| 整体板块资金净流入 |
| 尾盘10分钟（14:50后）无明显跳水 | 无主力集体出货迹象 |

**尾盘跳水判定：** 14:50后板块指数快速下跌超过 0.5% 视为跳水信号，降低参与意愿。

---

### 4. 资金持续性（排除一日游）

**直接排除以下板块：**

| 排除类型 | 判断依据 |
|---|---|
| 无逻辑消息刺激的题材 | 仅靠传言炒作，无政策/业绩/行业催化 |
| 资金无持续流入的冷门板块 | 近3日板块资金净流入为负 |
| 一日游题材 | 昨日曾大涨今日无延续性跟进 |

---

## 综合判断流程

```
Step 1: 提取当日板块涨幅 TOP3
Step 2: 候选标的是否归属其中？
  否 → 该标的淘汰
Step 3: 所在板块涨停数 ≥ 2？
  否 → 该板块淘汰
Step 4: 板块尾盘10分钟无跳水？
  有跳水 → 该板块降级处理
Step 5: 有无资金持续流入逻辑？
  否 → 排除，视为一日游
```

**全部通过 → 进入第三层个股筛选**
