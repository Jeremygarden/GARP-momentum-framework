# 铁律交易规则（纪律为王·决定收益）

**所有规则无条件执行，杜绝情绪化操作。**

---

## 一、入场规则

| 要素 | 规则 |
|---|---|
| **时间窗口** | 14:45 — 14:55（避开尾盘最后5分钟）|
| **单只仓位** | ≤ 15% |
| **总仓位** | ≤ 30%（最多同时持有2-3只）|
| **挂单策略** | 挂中间价成交，**不主动追高打卖一** |
| **选股数量** | 精选 1-3 只，不盲目扩大 |

> **14:55后严禁入场**——最后5分钟是主力出货窗口，成交价格失真风险极高。

---

## 二、离场规则（无条件执行·6种情景）

### 情景1：次日高开 ≥ 3%
```
操作：冲高时分批卖出（不追求最高点）
节奏：分2-3批，每批间隔2-5分钟
禁止：贪恋超额收益，继续持有
```

### 情景2：次日平开或小幅上涨（0~2%）
```
操作：9:30-9:45之间完成止盈
节奏：快速离场，不恋战
```

### 情景3：次日低开 ≥ 2%
```
操作：集合竞价阶段直接止损（9:25-9:30）
节奏：不等开盘，直接在集合竞价挂出卖单
禁止：抱有侥幸心理，等待反弹
```

### 情景4：次日低开 < 2%，盘中下跌
```
操作：跌破分时均价线2.5%且无反弹突破，无条件止损
节奏：触发条件后立即执行，不等待
时间限制：10:15前无论如何全部清仓
```

### 情景5：次日一字涨停或秒板
```
操作：唯一例外，可持有至涨停开板后卖出
节奏：开板后第一时间分批卖出
注意：若开板后快速回落，立即清仓
```

### 情景6：任何情况下的最终截止时间
```
次日10:15前，无论盈亏，全部清仓离场
无例外，无延迟
```

---

## 三、风控规则

| 触发条件 | 执行动作 |
|---|---|
| 当日累计亏损达 **3%** | 立即停止当日所有操作 |
| **连续2笔**交易亏损 | 强制空仓1个交易日，调整节奏 |
| 大盘破位（跌破MA20）| 无条件空仓观望 |
| 情绪退潮（高位股批量跌停）| 无条件空仓观望 |

---

## 四、屏蔽清单（规避90%亏损陷阱）

以下标的**永远不参与**，无论技术形态多好：

| 序号 | 屏蔽类型 | 判断标准 |
|---|---|---|
| 1 | 连续大涨高位透支股 | 近5个交易日累计涨幅 > 30% |
| 2 | 主力出货股 | 涨停后次日出现放量大阴线（跌幅 > 5%，量比 > 2）|
| 3 | 尾盘诱多股 | 14:55后急拉 > 2%，无资金持续支撑 |
| 4 | 孤狼股 | 脱离板块联动，独立走势，无同板块涨停跟随 |
| 5 | 基本面瑕疵股 | 大股东近期减持公告、业绩暴雷、商誉减值 |
| 6 | 弱势市场追涨股 | 大盘处于下跌趋势，任何形态达标均不参与 |

---

## 五、策略纪律检查清单（每次操作前自查）

```
入场前：
  □ 第一层大盘情绪是否全部通过？
  □ 板块热点是否明确，涨停数 ≥ 2？
  □ 个股六维条件全部达标？
  □ 资金大单净流入为正？
  □ 当前时间在14:45-14:55之间？
  □ 总仓位加上本次不超过30%？

离场前：
  □ 是否已到次日10:15？（到则无条件清仓）
  □ 开盘情景属于哪种？（对应执行规则）
  □ 当日是否已触发3%止损线？
  □ 是否连续2笔亏损（触发则强制空仓）？
```
